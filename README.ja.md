# Forza Telemetry Splitter

[English](README.md) · **日本語** · [Français](README.fr.md) · [Deutsch](README.de.md) · [Español](README.es.md)

Forza のテレメトリを複数のツールへ同時に送信します。

Forza Horizon 6 の「Data Out」テレメトリは、1つの IP とポートにしか送信できません。そのため、
[VirtualTCU](https://github.com/Forza-Love/fh6-virtual_tcu)（自動シフト）か、オートチューナーか、
ダッシュボードか——どれか1つを選ぶ必要がありました。

Forza Telemetry Splitter はその間に入ります。専用のポートで Forza のテレメトリを受け取り、各パケットを
そのまま、好きなだけ多くのローカルツールへ再送信します。遅延はミリ秒未満で、データは一切変更されない
ため、各ツールは Forza と直接通信しているのとまったく同じように動作します。

Turn 10、Playground Games、Microsoft とは提携しておらず、公認もされていません。「Forza」は Microsoft の
商標です。

## 主な機能

| 機能 | 説明 |
|------|------|
| 分配 | Forza のテレメトリを任意の数の宛先へ、パケットを変更せずに分配します。 |
| マルチゲーム対応 | Forza Horizon 4/5/6 と Forza Motorsport（7、2023）に対応。ゲームは自動検出され、アプリに表示されます。 |
| ステータスオーバーレイ | 画面右上の小さな表示で「接続済み / データなし」と、現在のギア・速度をひと目で確認できます。 |
| 多言語対応 | 日本語、英語、フランス語、ドイツ語、スペイン語。Windows の言語から自動選択（アプリ内で変更可）。 |
| トレイ常駐 | VirtualTCU と同様に、システムトレイで静かに動作します。 |
| 管理者不要 | ローカル UDP のみを扱うため、UAC のプロンプトは出ません。 |

## インストール

おすすめ — インストーラー:

1. [Releases](../../releases) ページから `ForzaTelemetrySplitterInstaller.exe` をダウンロードします。
2. ファイルを右クリック →「プロパティ」→「全般」タブ下部の「ブロックの解除」にチェック →「OK」。これで
   「Windows によって PC が保護されました」の画面を回避できます。
3. 実行します。インストーラーはユーザー単位のため、管理者プロンプトは出ません。デスクトップショート
   カットと「Windows 起動時に自動的に開始する」オプションを選べます。
4. 完了するとシステムトレイで起動します。

インストール不要派 / 上級者向け: 代わりに `ftsPortable.exe` をダウンロードして直接実行できます。ほとんどの方には
上記のインストーラーをおすすめします。

トレイアイコンが見つからない場合、Windows は新しいアイコンを既定で隠します。タスクバー右下の小さな
山型（^）をクリックすると表示されます。

## ゲーム内の設定

アプリは自動的に分配を開始します。あなたがやることは、Forza の送信先をこのアプリに向けるだけです。

1. トレイからアプリを開きます。ポート **44405** で待機し、VirtualTCU（通常の 5555）へ転送するよう
   既に設定されています。
2. お使いの Forza で Data Out を開きます（Horizon の場合: 設定 → HUD とゲームプレイ → Data Out。
   Motorsport にも同様の設定があります）:
   - Data Out: オン
   - IP アドレス: `127.0.0.1`
   - ポート: **`44405`**
   - パケット形式: Car Dash（Horizon）または Dash（Motorsport）
3. 他のツールはそのままで大丈夫です。このアプリが各ツールの通常ポートへ転送するため、設定変更は
   不要です。別のツールを追加するには、アプリで「追加」をクリックし、リストから選びます。
4. 走り出すと、右上の表示が緑（接続済み）になり、有効なすべてのツールが同時にデータを受け取ります。

### 既定のポート

| 内容 | IP | ポート |
|------|----|----|
| Forza Data Out → このアプリ（待機） | 127.0.0.1 | 44405 |
| VirtualTCU へ転送（変更不要） | 127.0.0.1 | 5555 |
| チューナーへ転送（例、既定では無効） | 127.0.0.1 | 9999 |

## 対応ツール

Forza のライブ UDP テレメトリを読み取るあらゆるツール（VirtualTCU、ForzaDash、SimHub、SIM Dashboard、
co-driver など）。各ツールには通常のポートへ転送するため、ツール側の変更は不要です。
